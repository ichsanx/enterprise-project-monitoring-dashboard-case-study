# Upload Instructions

1. Open your GitHub repository.
2. Click **Add file** → **Upload files**.
3. Extract the ZIP file on your computer first.
4. Upload `README.md` and the entire `docs` folder.
5. Choose **Commit directly to the main branch**.
6. Use commit message: `Complete case study documentation and visuals`.
7. Click **Commit changes**.

Important:
- This package uses anonymized portfolio recreations and synthetic values.
- Do not upload original internal source code, credentials, customer data, private URLs, or confidential documents.
